# Justinian-Digest-project
Data and code for processing the Latin-Italian parallel corpus of Justinian Digest, in order to extract a digital bilingual glossary of Latin legal lexicon.

Reference publication: Marongiu, P. & Sassolini, E. (2026). From Lemmatization to Legal Terminology: Assessing an Hybrid Pipeline on Justinian’s Digest. LT4HALA 2026. 

Website: https://dbtvm1.ilc.cnr.it/digesto/
